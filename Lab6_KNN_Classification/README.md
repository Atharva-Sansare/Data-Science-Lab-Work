# Lab 6: KNN Classification

## Description
This lab demonstrates the use of the K-Nearest Neighbors (KNN) algorithm for classification tasks.

## Objective
The objective of this lab is to understand how KNN works and how it can be used to classify data points based on similarity.

## Dataset
- Dataset used for classification
- Contains features and class labels

## Steps Performed
1. Imported required libraries (Pandas, NumPy, Scikit-learn)
2. Loaded and explored the dataset
3. Split data into training and testing sets
4. Applied KNN algorithm
5. Chose appropriate value of K
6. Evaluated model performance

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Results
- Successfully classified data using KNN
- Achieved reasonable accuracy

## Conclusion
KNN is a simple and effective classification algorithm based on distance between data points.
