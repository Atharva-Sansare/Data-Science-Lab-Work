# Customer Segmentation using Clustering

## Description
This project groups customers based on their behavior using K-Means clustering.

## Dataset
- Clustering_Class.xlsx
- Contains customer data for segmentation

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Data Visualization
4. Model Building using K-Means Clustering

## Results
Customers are divided into different clusters based on their patterns.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Conclusion
This project helps in understanding customer groups, which can be useful for targeted marketing.
