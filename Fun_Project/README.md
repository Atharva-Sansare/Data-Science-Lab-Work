# Fun Project: Number Guessing Game

## Description
This is a simple interactive Python game where the user tries to guess a randomly generated number within a given range. The program provides feedback after each guess to guide the user.

## Objective
The goal of this project is to demonstrate basic programming concepts such as loops, conditional statements, and user input handling.

## How It Works
- The system generates a random number between a fixed range (e.g., 1 to 100)
- The user enters a guess
- The program checks the guess and gives feedback:
  - "Too high" if the guess is greater than the number
  - "Too low" if the guess is smaller
  - "Correct" if the guess matches the number
- The loop continues until the correct number is guessed

## Features
- Random number generation
- User interaction using input
- Conditional logic for feedback
- Looping until correct answer

## Tools Used
- Python
- Random module

## Results
The user successfully guesses the correct number through multiple attempts with guided feedback.

## Conclusion
This project helps in understanding fundamental programming concepts like loops, conditions, and user interaction in Python.
